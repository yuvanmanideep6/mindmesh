# MindMesh AI Chatbot (WIP)
A full-stack AI chatbot web application built using React, Node.js, Express, and MongoDB.

## Features
* AI-powered chat system
* Workspace-based conversations
* Chat history stored in MongoDB
* JWT authentication
* Dynamic UI with typing effects

## Tech Stack
* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB Atlas
* AI Integration: Gemini API

## Status
Work in Progress – Backend integration and deployment in progress

## How to Run
### Backend
cd server
npm install
npm run dev
### Frontend
cd client
npm install
npm run dev
